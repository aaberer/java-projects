DecimalToBinary Class Usage: java cs250.ec.convert.DecimalToBinary <decimal-number>
    Takes a decimal number input between 0 and 1 and outputs the corresponding binary base 2 representation

AlphaHistogram Class Usage: java cs250.ec.counting.AlphaHistogram "<sentence>"
    Takes a string and counts the number of letter occurences, not case sensitive.

Equation Class Usage: java cs250.ec.order.Equation <x> <y>
    Enter two ints x,y and will convert to decimal rounded to 5 decimal places.
